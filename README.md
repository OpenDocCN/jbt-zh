## 下载

### Docker

```
docker pull apachecn0/jbt-zh
docker run -tid -p <port>:80 apachecn0/jbt-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g jbt-zh
jbt-zh <port>
# 访问 http://localhost:{port} 查看文档
```
